Do you know?

  • You can move a button on a tab in the main application window to a new location by simultaneously holding down the Ctrl key and the left mouse button.

  • By holding down the Ctrl key and the left mouse button at the same time, you can also change the order of the tabs in the main application window.

  • If you drag a button onto another existing button, the buttons will swap places.

  • Left-clicking on the application's tray icon opens and hides the main application window.
