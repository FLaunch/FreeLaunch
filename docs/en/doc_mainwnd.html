Interface elements:

  1. Tabs
    You can customize the number of tabs and rename each tab.

  2. Tab content
    You can place any number of shortcuts on multiple tabs.

  3. Button information
    Displays a description of the button that the mouse cursor is currently over.

  4. Date and time
    Shows the current date and time.

Hotkeys:

  1. Alt + Left Mouse Button
    Holding this hotkey on any button moves the main application window.

  2. Ctrl + Left Mouse Button
    Holding this hotkey on any button moves selected button or selected tab.

  3. F1
    Pressing this hotkey opens the help file.

  4. F2
    Pressing this hotkey renames the currently open tab.

  5. Ctrl + Q
    Pressing this hotkey terminates the application.

  6. Ctrl + S
    Pressing this hotkey opens the settings window.

  7. Alt + F4
    Pressing this hotkey hides the application's main window.

  8. Ctrl + Win
    Pressing this global hotkey (not only in an application) shows or hides the application's main window.
