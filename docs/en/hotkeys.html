Global hotkeys
Ctrl + Win Show FreeLaunch main window.
Main window
Alt + F4 Hide FreeLaunch main window.
Alt + Left Mouse Button
(pressed on any button) 		Move the main application window.
Ctrl + Left Mouse Button
(pressed on any button) 		Move selected button or selected tab.
F1 Open help file.
F2 Rename the selected tab in the main application window.
Ctrl + N Add new tab.
Ctrl + Q Terminate the application.
Ctrl + S Open application settings window.
Ctrl + W Delete selected tab.
Settings window
Alt + F4 Close application settings window.
Enter Press «Apply» button.
Ctrl + Enter Press «OK» button.
Esc Press «Cancel» button.
Other windows
Alt + F4 Close the current window.
Enter Press «OK» button.
Esc Press «Cancel» button.
