Interface options:

  1. Language
    Sets the interface language of the application.

  2. Theme
    Sets the application's interface theme.

  3. Window title style
    Sets the title style of the application's main window.

  4. Tab style
    Sets the style of tabs in the main application window.

  5. Number of tabs
    Sets the number of tabs in the main application window.

  6. Number of rows on the tab
    Sets the number of rows per tab.

  7. Number of columns on the tab
    Sets the number of columns per tab.

  8. Padding between buttons
    Sets the amount of padding between buttons on a tab.

  9. Button size
    Button sizes on a tab.

    • Width
      Sets the width of a button on a tab.

    • Height
      Sets the height of a button on a tab.
