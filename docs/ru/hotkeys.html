Глобальные горячие клавиши
Ctrl + Win Показать главно окно FreeLaunch.
Главное окно
Alt + F4 Скрыть главное окно FreeLaunch.
Alt + Left Mouse Button
(нажатая на любой кнопке) 		Переместить главное окно приложения.
Ctrl + Left Mouse Button
(нажатая на любой кнопке) 		Переместить выбранную кнопку или выбранную вкладку.
F1 Открыть файл справки.
F2 Переименовать выбранную вкладку в главном окне приложения.
Ctrl + N Добавить новую вкладку.
Ctrl + Q Завершить работу приложения.
Ctrl + S Открыть окно настроек приложения.
Ctrl + W Удалить выбранную вкладку.
Окно настроек
Alt + F4 Закрыть окно настроек.
Enter Нажать кнопку «Применить».
Ctrl + Enter Нажать кнопку «OK».
Esc Нажать кнопку «Отмена».
Другие окна
Alt + F4 Закрыть текущее окно.
Enter Нажать кнопку «OK».
Esc Нажать кнопку «Отмена».
